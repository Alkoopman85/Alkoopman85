# Alexander Levin-Koopman

Data scientist and applied data science lecturer building deeper into practical AI engineering.

I like taking technical systems apart, rebuilding the important pieces, and explaining how they work from the inside out. My background is in mathematics, applied data science, machine learning, and teaching — and my current focus is on practical LLM systems: RAG, embeddings, agent harnesses, evaluation, tool use, and the infrastructure that makes AI systems useful.

## Current focus

- Practical AI engineering for data scientists
- Retrieval-augmented generation (RAG) and retrieval evaluation
- Embeddings, vector search, and semantic retrieval
- Agent harnesses, tool calling, memory, and context management
- MCP-based research workflows and AI tooling
- From-scratch explanations of ML/AI systems

## What I'm building toward

I'm using this as a place to collect older data science work, current learning projects, and future AI engineering builds in a way that is useful to revisit — not just as a portfolio, but as a technical record of what I am learning and building.
